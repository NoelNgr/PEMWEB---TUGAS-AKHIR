### Konsep Utama: public/ vs src/

1.  **Folder `public/` (Area Tampilan)**
    * Ini adalah semua bagian yang diakses langsung oleh browser atau pengguna (user).
    * Ini adalah "wajah" dari website Anda.
    * Isinya: Halaman .php yang dilihat user (login.php, dashboard.php), semua file CSS, JS, dan Gambar.

2.  **Folder `src/` (Area Logika)**
    * Ini adalah "otak" dari website Anda. Area ini TIDAK BOLEH diakses langsung oleh browser.
    * Isinya: Koneksi database, file pemroses form (actions), dan data internal (data).

### Rincian Asal-Usul File (Lama -> Baru)

#### ðŸ“‚ `src/` (Logika Backend)

* **File Baru:** `src/conn.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/conn.php`

* **File Baru:** `src/actions/handle_login.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/LOGIN/proses_login.php`

* **File Baru:** `src/actions/handle_register.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/REGISTER/proses_register.php`

* **File Baru:** `src/actions/handle_profile_update.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/DASHBOARD/update_profile.php`

* **File Baru:** `src/actions/handle_photo_upload.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/DASHBOARD/upload_foto.php`

* **File Baru:** `src/actions/handle_cart.php`
    * **Asal:** BARU. Logika dari `DASHBOARD/cart.php` dan logika keranjang dari `DASHBOARD/dashboard.php` digabung ke file ini.

* **File Baru:** `src/actions/handle_checkout.php`
    * **Asal:** BARU. Berisi kode PHP (logika) yang dipisah dari `CHECKOUT/proses_checkout.php`.

* **File Baru:** `src/actions/logout.php`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/logout.php`

* **File Baru:** `src/data/products.php`
    * **Asal:** BARU. Berisi array `$produkTerlaris` yang dipindah dari `DASHBOARD/dashboard.php`.

* **File Baru:** `src/data/orders.json`
    * **Asal:** `PEMWEB---TUGAS-AKHIR/CHECKOUT/orders.json`
